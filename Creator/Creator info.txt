Library for creating Services, Repositories directories and classes(Repositories interfaces, Service interfaces, Services implementation) and application.properties file for Spring. Call static method Creator.create with parameters StrategyType and Main class. StrategyType is enumeration. You can choose in four options: REPOSITORIES_AND_SERVICES, SERVICES, REPOSITORIES, APPLICATION_PROPERTIES. Scan all folders and file on level and below Main class for Entity annotation. Based on these files create directories and classes.

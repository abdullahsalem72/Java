Library for creating Services, Repositories, Controllers directories and classes(Repositories interfaces, Service interfaces, Services implementation, Controllers). Call static method Creator.create with parameters StrategyType, entity package path(getPath), destination package path(same path. StrategyType is enumeration. You can choose in four options: REPOSITORIES_AND_SERVICES, SERVICES, REPOSITORIES, CONTROLLERS. Scan entity folder for Entity annotation. Based on these files create directories and classes.
